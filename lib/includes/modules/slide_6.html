CommonJS Module Pattern

  • Mediante CommonJS cada fichero JavaScript almacena módulos en el contexto común module.
  • Evita el problema de global namespace pollution
  • Las dependencias se indican de manera explícita
  • Sintaxis clara
  

var users = ['mohamed', 'karim', 'amine'];

module.exports = {
showUsers: function(){
users.map(function(user){
console.log(user);
});
},
addUser: function(name){
users.push(name);
}
}
