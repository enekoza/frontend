¿Por qué usar módulos?


Los principales beneficios derivados del uso de módulos son:

Mantenibilidad

  • Un módulo es autocontenido.
  • Minimiza las dependencias entre diferentes partes del código fuente.
  • Actualizar un módulo es mas sencillo.

Namespacing

  • En JavaScript, las variables fuera del scope de una función son globales.
  • "namespace pollition"
  • Los módulos permiten crear scopes privados para nuestras variables.

Reusabilidad

  • Evita el copy-paste de funciones entre proyectos.
  • Mediante un módulo es mas sencillo el reutilizarlo en diferentes desarrollos.
