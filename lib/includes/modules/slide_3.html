Anonymous closure

  • La función anónima dispone de su propio entorno de evaluación. Closure
  • Se evalua inmediatamente.
  • Oculta las variables del namespace padre (global)
  

// global function
function showUsers(users){
users.map(function(user){
console.log(user);
});
}

(function () {
// body of our module
var users = ['mohamed', 'karim', 'amine'];
showUsers(users);
})();
