Webpack

  • Webpack es un module bundler

  • Es una herramienta de agregación de recursos para aplicaciones web que permite generar una distribución única a partir de un conjunto establecido de assets.

  • Entre los recursos que es capaz de gestionar webpack, podemos encontrar formatos soportados por defecto como: HTML, JavaScript, CSS, así como otros formatos que necesitan ser transformados como SASS, CoffeeScript o Jade.

  • Obtendremos un único fichero empaquetado y minificado con todos los recursos necesarios para nuestro desarrollo.
