ES6 Modules

  • Similar a CommonJS, sintaxis sencilla, preferentemente exports únicos y soporte para dependencias cíclicas.
  • Similar a AMD, soporte para carga asíncrona y caga de módulos configurable.
  • Sintaxis mas compacta qeu CommonJS
  • Puden ser analizados de manera estática (comporobación estática, optimización...)
    

//------ lib.js ------
export const sqrt = Math.sqrt;
export function square(x) {
return x * x;
}
export function diag(x, y) {
return sqrt(square(x) + square(y));
}

//------ main.js ------
import { square, diag } from 'lib';
console.log(square(11)); // 121
console.log(diag(4, 3)); // 5
